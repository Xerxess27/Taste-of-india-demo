# Taste of India — website concept

A responsive, bilingual restaurant website prototype for Taste of India in Neuenstadt am Kocher.

## Preview

Run a local server from this folder:

```powershell
npm.cmd run start
```

Then open `http://127.0.0.1:4173`.

Run a syntax sanity check with:

```powershell
npm.cmd run check
```

## Included

- Premium responsive design
- New restaurant logo assets and favicon from the provided logo image
- German/English language switch
- Live Berlin-time open/closed status
- Full 120-item menu from the current online ordering menu
- 120 generated, optimized menu photos in `assets/menu/`
- Premium scroll reveals, subtle hero parallax, steam/spice ambience, and ripple micro-interactions
- Category tabs, skeleton menu loading, menu search, sold-out state, popular badges, ratings, and cart drawer
- App-like mobile bottom navigation and scroll-to-top control
- Daily special, customer review cards, and embedded Google Map
- Reservation request UI with Tuesday validation
- Lunch offer, contact details, directions, delivery/order handoff
